อุปกรณ์ IoT ที่ใช้งาน

ถุงมือการกายภาพบำบัดสำหรับการฟื้นฟูสมรรถภาพร่างกาย

SP32 คือ wifi microcontroller ที่ถูกพัฒนาต่อจาก ESP8266 โดยเพิ่ม CPU เป็น 2 core, Wi-Fi ที่เร็วขึ้น, มีขา GPIO ให้ใช้งานมากขึ้น และรองรับ Bluetooth อีกด้วย นอกจากนี้ ESP32 ยังมาพร้อมกับ touch-sensitive pins ที่สามารถใช้ปลุก ESP32 จากโหมด deep sleep และยังมี hall effect sensor และ temperature sensor ในตัว

DHT22 หรือ AM2302 โมดูลวัดอุณหภูมิและความชื้นแม่นยำสูง

เซนเซอร์วัดอัตราการเต้นของหัวใจเป็นอุปกรณ์ตรวจสอบส่วนบุคคลที่ช่วยให้สามารถวัด / แสดงอัตราการเต้นของหัวใจแบบเรียลไทม์หรือบันทึกอัตราการเต้นของหัวใจสำหรับการศึกษาในภายหลัง
